# InstallerFileFormat
A Installer System Similar To GIT

# Usage
### Downloading
```
java -f:<link to valis iff file>
java -jar iff.jar -f:https://github.com/ChefZander/repo/raw/main/example.iff
```
If the URL is not valid, it will throw an error.
